# Рубрика оценивания КИМ 1.1

Оценивание проводится по единой 10-балльной шкале ФОС.

## 1. Критерии оценивания

| Критерий | Баллы |
| :--- | :--- |
| **Постановка задачи и корректное определение типа обратной связи** | 1.0 |
| **Подготовка данных, построение матрицы взаимодействий, расчёт разреженности** | 1.5 |
| **Проведение EDA (распределения, визуализация, анализ активности и популярности)** | 2.0 |
| **Реализация и оценка baseline-моделей (Global, User, Item, Popularity)** | 1.5 |
| **Анализ популярностного смещения и фильтрующих пузырей** | 1.5 |
| **Анализ ограничений данных (cold start, sparsity, ограничения обратной связи)** | 1.5 |
| **Качество выводов, обоснованность суждений** | 1.0 |
| **Итого** | **10.0** |

## 2. Уровни достижения результата

| Уровень | Баллы | Описание |
| :--- | :--- | :--- |
| **Не достигнут** | 0–3 | Задание не выполнено или выполнено формально. Отсутствует EDA, матрица не построена, смещения не анализируются, baseline не реализован. |
| **Базовый** | 4–5 | Данные загружены, матрица построена, но EDA поверхностное. Реализован один простой baseline. Анализ смещений фрагментарный. |
| **Средний** | 6–7 | Проведено полное EDA с визуализациями. Реализованы все основные baseline-модели. Проведён анализ популярностного смещения и разреженности. |
| **Высокий** | 8–9 | Реализован полный объём работы. Присутствует глубокий анализ смещений (popularity bias, фильтрующие пузыри), рассмотрены ограничения, сделаны обоснованные выводы. |
| **Продвинутый** | 10 | Работа носит исследовательский характер. Предложены количественные метрики для оценки смещений. Сформулированы гипотезы и рекомендации по устранению выявленных проблем. |

## 3. Типовые основания для снижения баллов

- Неверно определён тип обратной связи (explicit/implicit).
- Матрица взаимодействий построена некорректно (строки/столбцы перепутаны).
- Отсутствуют визуализации распределений.
- Baseline не сравниваются между собой.
- Анализ популярностного смещения сводится к констатации факта без количественной оценки.
- Не обсуждаются фильтрующие пузыри.
- Не рассматриваются ограничения наблюдаемой обратной связи.
- Выводы не соответствуют полученным данным.
